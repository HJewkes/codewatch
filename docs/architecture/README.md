# Architecture maps

Self-contained interactive HTML maps of the codewatch codebase, generated by `code-style graph index` + `code-style graph render`. Open with file:// — no backend, no CDN, no install.

| File | Subject | Files | Nodes | Edges |
|---|---|---|---|---|
| [`codewatch.html`](codewatch.html) | Full monorepo (post-Phase-4 substrate) | 172 | 369 | 520 |
| [`core.html`](core.html) | `@code-style/core` substrate alone — the package the upcoming graph mode and visualization tools depend on | 21 | 51 | 51 |

## Controls

- **Click** a node to pin its details to the side panel (full id, kind, attrs).
- **Hover** to highlight the node's neighborhood.
- **Search** the input top-right to filter by id substring (non-matches fade).
- **Zoom + pan** with scroll wheel and drag.

## Regenerate

```bash
pnpm -r build
node packages/cli/dist/index.js graph index packages
node packages/cli/dist/index.js graph render \
  --db packages/.codewatch/graph.db \
  --out docs/architecture/codewatch.html \
  --title "codewatch architecture"
```

`.codewatch/` is gitignored; the HTML is the artifact worth keeping.
